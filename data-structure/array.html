array是js的内置对象，提供了增删查改的api，不需要我们自己去实现
  1. 声明数组:
    var arr = ['a', 'b']
  2. 查询:(通过索引访问)
    arr[0] arr[1]
  3. 遍历:
    arr.forEach(function (item, index, array) { console.log(item, index); });
  4. 增加:
    arr.push('c') // 添加到末尾 arr.unshift('d') // 添加到头部
  5. 修改:
    arr.splice(pos, 1, 'u') // 修改某个指定位置的元素
  6. 删除:
    arr.pop() // 删除末尾元素 arr.shift() // 删除头部元素 arr.splice(pos, 1) // 从指定位置删除某个元素
